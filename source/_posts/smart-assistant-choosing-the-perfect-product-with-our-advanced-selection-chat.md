## Product Picker Dialog

 This dialog allows you to obtain the Product Code or Upgrade Code of an application. You can obtain it from an installed application on your machine or from an MSI package.

![Product picker](https://cdn.advancedinstaller.com/img/dialog/product-picker.png "Product picker")  

![Note](https://cdn.advancedinstaller.com/svg/common/IconMessageNote.svg)You can obtain the Product Code or Upgrade Code only for MSI-based installations.
